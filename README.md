# Coursework — расчёт числа зубьев передачи

Настольное приложение на Python для **расчёта числа зубьев шестерни и колеса** цилиндрической передачи по методичке:

- расчёт модуля по формуле (6.8);
- учёт выносливости на изгиб по формулам (6.9), (6.10), (6.11);
- использование табличных данных по материалам, допускаемым напряжениям и стандартным модулям, сохранённых в базе данных PostgreSQL.

Графический интерфейс реализован на `tkinter`.  
Табличные данные хранятся в PostgreSQL и загружаются из дампа `db/gearcalc_schema.sql`.

---

## 1. Что нужно, чтобы запустить

Предполагается **чистый компьютер**, на котором нет ничего, кроме операционной системы (Windows).

Минимальные требования:

1. **Python**: версия **3.10 или новее**  
   (разрабатывалось и тестировалось на Python 3.13 для Windows).
2. **PostgreSQL**: версия **13 или новее**.
3. Свободный порт для PostgreSQL (по умолчанию в проекте используется `5432`).

Больше ничего ставить не нужно — `tkinter` входит в стандартную библиотеку Python на Windows, а остальные зависимости устанавливаются через `pip`.
